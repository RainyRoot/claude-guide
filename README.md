# claude-guide

A practical guide to using Claude properly. Covers the chat app, Claude Code, how to pick a model, skills, subagents, project setup and a bunch of other things that took me a while to figure out.

Written for people who know how to use a computer and GitHub but are new to Claude specifically.

## if you're completely new

1. Read the [Overview](docs/01-overview/index.md) first, takes 2 minutes.
2. Go to [claude.ai](https://claude.ai) and try it. No install needed.
3. When you want to use it for coding, come back for [Claude Code](docs/04-claude-code/index.md).
